A JetBrains plugin designed to monitor SQL Server databases directly from your favorite JetBrains IDEs, such as DataGrip or IntelliJ IDEA Ultimate.

[Link to demo video](https://youtu.be/OEbcTEhJUsg?si=2Di-QUwZzPaxyOCu)

## üöÄ Features

- Monitor SQL Server databases from data sources defined in JetBrains IDEs (DataGrip, IntelliJ IDEA Ultimate, etc.).
- View real-time performance metrics such as :
    - CPU usage in percent
    - Waiting tasks
    - I/O usage in MB per second
    - Batch requests per second
    - Total TempDB size in MB
    - RAM usage in MB
